Enable IN channel to start transfer.

Parameters
chNum: Select channel.
transferLen: transfer data length.
maxPackSize: The Max sizeof Packet to transfer.
dataPid: Channel data PID. This value can be one of the following values:
  • DATA_0 : DATA0.
  • DATA_1 : DATA1.
  • DATA_2 : DATA2.
  • MDATA : Controled transmission.
  • SETUP : UnControled transmission.
epType: Endpoint type. This value can be one of the following values:
  • EP_TYPE_CONTROL : Control type.
  • EP_TYPE_ISO : Isochronous type.
  • EP_TYPE_BULK : Batch type.
  • EP_TYPE_INTERRUPT : Interrupt type.
Return values
None
