Read the IN endpoint interrupt status.

Parameters
ep: IN endpiont number
flag: IN ep interrupt flag. This parameter can any combination of the following value:
  • USB_INT_EP_IN_TSFCMP : Transfer completed.
  • USB_INT_EP_IN_EPDIS : Endpoint disabled.
  • USB_INT_EP_IN_TO : Timeout.
  • USB_INT_EP_IN_ITXEMP : IN token received when TxFIFO is empty.
  • USB_INT_EP_IN_IEPNAKE : IN endpoint NAK effective.
  • USB_INT_EP_IN_TXFE : TxFIFO empty.
  • USB_INT_EP_IN_ALL : all interrupt above.
Return values
returnOut endpoint interrupt status
