◆ BAKPR_ConfigRTCOutput()

void BAKPR_ConfigRTCOutput ( BAKPR_RTC_OUTPUT_SOURCE_T  soure)

Select the RTC output source to output on the Tamper pin.

Parameters
sourespecifies the RTC output source. This parameter can be one of the following values:
  • BAKPR_RTC_OUTPUT_SOURCE_NONE : no RTC output on the Tamper pin.
  • BAKPR_RTC_OUTPUT_SOURCE_CALIBRATION_CLOCK: output the RTC clock with frequency divided by 64 on the Tamper pin.
  • BAKPR_RTC_OUTPUT_SOURCE_ALARM : output the RTC Alarm pulse signal on the Tamper pin.
  • BAKPR_RTC_OUTPUT_SOURCE_SECOND : output the RTC Second pulse signal on the Tamper pin.
Return values
None
