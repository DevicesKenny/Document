◆ I2C_ClearIntFlag()

void I2C_ClearIntFlag ( I2C_T *  i2c,
uint32_t  flag 
)

Clears the I2C interrupt flag bits.

Parameters
i2cI2C selet 1 or 2
flagspecifies the I2C flag The parameter can be any combination of the following values:
  • I2C_INT_FLAG_SMBALT: SMBus Alert flag
  • I2C_INT_FLAG_TTE: Timeout or Tlow error flag
  • I2C_INT_FLAG_PECE: PEC error in reception flag
  • I2C_INT_FLAG_OVRUR: Overrun/Underrun flag (Slave mode)
  • I2C_INT_FLAG_AE: Acknowledge error flag
  • I2C_INT_FLAG_AL: Arbitration lost flag (Master mode)
  • I2C_INT_FLAG_BERR: Bus error flag
Return values
None
Note
1)I2C_INT_FLAG_STOP: Stop detection flag is cleared by software sequence: a read operation to I2C_STS1 register (I2C_ReadIntFlag()) followed by a write operation to I2C_CRTL1 register (I2C_Enable()). 2)I2C_INT_FLAG_ADDR10: 10-bit header sent flag is cleared by software sequence: a read operation to I2C_STS1 (I2C_ReadIntFlag()) followed by writing the second byte of the address in I2C_DATA register. 3)I2C_INT_FLAG_BTC: Byte transfer complete flag is cleared by software sequence: a read operation to I2C_STS1 register (I2C_ReadIntFlag()) followed by a read/write to I2C_DATA register (I2C_TxData()). 4)I2C_INT_FLAG_ADDR: Address sent flag is cleared by software sequence: a read operation to I2C_STS1 register (I2C_ReadIntFlag()) followed by a read operation to I2C_STS2 register ((void)(I2Cx->STS2)). 5)I2C_INT_FLAG_START: Start bit flag is cleared software sequence: a read operation to I2C_STS1 register (I2C_ReadIntFlag()) followed by a write operation to I2C_DATA register (I2C_TxData()).
