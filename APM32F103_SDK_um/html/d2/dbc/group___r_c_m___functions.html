void RCM_ConfigPLL ( RCM_PLLSEL_T  pllSelect,
RCM_PLLMF_T  pllMf 
)

Configures the PLL clock source and multiplication factor.

Parameters
pllSelectPLL entry clock source select This parameter can be one of the following values: For APM32F105xx and APM32F107xx devices:
  • RCM_PLLSEL_HSI_DIV_2: HSI clock divided by 2 selected as PLL clock source
  • RCM_PLLSEL_PREDIV1 : PLL prescaler 1 clock selected as PLL clock source For other devices:
  • RCM_PLLSEL_HSI_DIV_2: HSI clock divided by 2 selected as PLL clock source
  • RCM_PLLSEL_HSE : HSE clock selected as PLL clock source
  • RCM_PLLSEL_HSE_DIV2 : HSE clock divided by 2 selected as PLL clock source
pllMfPLL multiplication factor For APM32F105xx and APM32F107xx devices: This parameter can be one of the following values:
  • RCM_PLLMF_4 : PLL Multiplication Factor Configures to 4
  • RCM_PLLMF_5 : PLL Multiplication Factor Configures to 5
  • RCM_PLLMF_6 : PLL Multiplication Factor Configures to 6
  • RCM_PLLMF_7 : PLL Multiplication Factor Configures to 7
  • RCM_PLLMF_8 : PLL Multiplication Factor Configures to 8
  • RCM_PLLMF_9 : PLL Multiplication Factor Configures to 9
  • RCM_PLLMF_6_5: PLL Multiplication Factor Configures to 6.5 For other devices: This parameter can be RCM_PLLMF_x where x can be a value from 2 to 16.
Return values
None
Note
PLL should be disabled while use this function.
